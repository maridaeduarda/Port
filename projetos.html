Cada projeto teve como objetivo melhorar minhas habilidades em lógica e desenvolvimento.

LISTA DE PROJETOS

  • Criação de uma página simples em HTML.
  • Simulação de cadastro de clientes e funcionários.
  • Desenvolvimento de um portfólio.
  • Exercícios de estruturação de páginas
  • Desenvolvimeto de um site para uma cliente.

DESCRIÇÃO DOS PROJETOS

Página HTML simples: Primeiro contato com criação de páginas e uso de títulos, textos e listas.

Cadastro de Clientes e Funcionários: Atividades práticas simulando um sistema de cadastro.

Portfólio pessoal: Site criado para apresentar informações pessoias e acadêmicas.

Site para Cliente: Atualmente estou aprendendo e deselvolvendo um site real para uma cliente, colocando em prática meus conhecimentos e aprendizados do curso e apredendo novas formas de organizar e estruturar páginas.

APRENDIZADOS

Esses projetos estão sendo muito importantes para meu conhecimento na área da tecnologia. Principalmente o deselvolvimento do site para cliente, que está me ajudando a entender melhor como funciona o projeto real.

Além disso, aprendemos a criar um banco de dados de clientes e funcionários.
